✂️ Geometric Operations: Resize, Crop, Rotate, Flip

📏 Resize
cv2.resize(img, (width, height))
resized = cv2.resize(img, (640,480), interpolation=cv2.INTER_AREA)
Preserve aspect ratio with fx/fy.
✂️ Crop
img[y1:y2, x1:x2]
cropped = img[50:250, 100:300]
Numpy slicing = ROI extraction.
🔄 Rotate
cv2.getRotationMatrix2D + warpAffine
(h,w) = img.shape[:2]; M = cv2.getRotationMatrix2D((w/2,h/2), 45, 1.0); rotated = cv2.warpAffine(img, M, (w,h))
🪞 Flip
cv2.flip(img, flipCode)
flip_horiz = cv2.flip(img, 1) # 1 = horizontal, 0 = vertical, -1 = both
💡 Real‑world tip: When building a document scanner, always resize to a standard width first, then crop the region of interest (ROI) to reduce noise.

🌀 Image Filtering & Edge Detection

🧽 Blurring (Noise Reduction)
  • GaussianBlur: cv2.GaussianBlur(img, (5,5), 0)
  • Median blur: great for salt‑pepper noise
  • Bilateral: preserves edges
gauss_blur = cv2.GaussianBlur(image, (7,7), 1.5)
🔪 Sharpening (Enhance edges)
  • Kernel: [[0,-1,0], [-1,5,-1], [0,-1,0]]
  • Apply using cv2.filter2D()
kernel_sharpen = np.array([[0,-1,0],[-1,5,-1],[0,-1,0]])
sharp = cv2.filter2D(img, -1, kernel_sharpen)
⚡ Edge Detection (Sobel & Canny)
Sobel Operator
sobel_x = cv2.Sobel(gray, cv2.CV_64F, 1, 0, ksize=3)
sobel_combined = cv2.magnitude(sobel_x, sobel_y)
Gradient magnitude — direction matters.
Canny Edge Detector
edges = cv2.Canny(gray, 50, 150)
# Lower/upper thresholds control sensitivity
edges = cv2.Canny(blurred, 30, 100)
Gold standard for edge detection: low noise + clear boundaries.

🎚️ Thresholding & Binary Images

Simple threshold: convert grayscale to binary (black & white) using a fixed cutoff value.

_, binary = cv2.threshold(gray, 127, 255, cv2.THRESH_BINARY)

Adaptive Thresholding — handles uneven lighting (crucial for photos of documents taken with shadows).

adaptive_thresh = cv2.adaptiveThreshold(gray, 255, cv2.ADAPTIVE_THRESH_GAUSSIAN_C, cv2.THRESH_BINARY, 11, 2)
🌓 Adaptive Mean
ADAPTIVE_THRESH_MEAN_C Threshold = mean of neighborhood − constant
🌟 Gaussian Adaptive
ADAPTIVE_THRESH_GAUSSIAN_C Weighted sum of neighborhood pixels
📄 Why it matters for scanners: documents clicked under desk lamp have bright center and dark corners. Adaptive thresholding produces clean binary output ready for OCR!

📄✨ Mini Project: Document Scanner Prep

🔍 Goal: Clean a photo of a document

Pipeline: Grayscale → Gaussian Blur → Edge Detection (Canny) → Adaptive Thresholding → final binary clean image ready for contour detection or OCR.

# Complete Document Scanner Prep (Stage 1)
import cv2
import numpy as np

def prepare_document(image_path):
   img = cv2.imread(image_path)
   gray = cv2.cvtColor(img, cv2.COLOR_BGR2GRAY)
   # reduce noise while preserving edges
   blurred = cv2.GaussianBlur(gray, (5,5), 0)
   # detect strong edges (used for contour finding later)
   edges = cv2.Canny(blurred, 50, 150)
   # adaptive threshold for clean binary map
   binary_clean = cv2.adaptiveThreshold(blurred, 255, cv2.ADAPTIVE_THRESH_GAUSSIAN_C, cv2.THRESH_BINARY, 11, 2)
   return binary_clean, edges

result_binary, edge_map = prepare_document("receipt_photo.jpg")
cv2.imwrite("scanned_ready.png", result_binary)
✅ Step-by-step expected output
1. Grayscale conversion removes color noise.
2. Gaussian blur reduces high-frequency noise.
3. Canny detects page boundaries.
4. Adaptive threshold produces uniform text regions.

📝 Key Takeaways & Best Practices

✅ Always denoise before edge detection
Gaussian blur reduces false edges from camera sensor noise.
🎯 Adaptive threshold for lighting variation
Don't rely on fixed threshold for real-world documents.
⚙️ Canny thresholds tune via trial
Lower ratio (1:2 or 1:3) between low/high thresholds works well.
