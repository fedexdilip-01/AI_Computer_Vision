# AI_Computer_Vision


### 7-Day AI Computer Vision Program 

## Day 1: Foundations & Image Basics

# Topics
What is computer vision?
Pixels, color spaces (RGB, Grayscale, HSV)
Loading, displaying, and saving images

# Tools
OpenCV, Matplotlib, NumPy

# Practice

Convert RGB ↔ Grayscale ↔ HSV
Split and merge color channels

#Mini Project
Image Color Extractor – Extract dominant color channel and display histograms.</br>

## Day 2: Image Processing & Transformations
Topics

Resizing, cropping, rotating, flipping

Image filtering (blur, sharpen, edge detection)

Thresholding & binary images

Practice

Apply Gaussian blur, Sobel/Canny edges

Adaptive thresholding for uneven lighting

Mini Project
Document Scanner Prep – Clean a photo of a document (grayscale → blur → edge detection → threshold).

Day 3: Feature Detection & Matching
Topics

Corners (Harris, Shi-Tomasi)

Keypoint detectors (SIFT, ORB)

Feature matching (BFMatcher, FLANN)

Practice

Detect corners in a checkerboard

Match keypoints between two similar images

Mini Project
Panorama Stitcher – Stitch two overlapping images using ORB + homography.

Day 4: Introduction to Deep Learning for Vision
Topics

Why CNNs?

Convolution, pooling, activation functions

Loading pre-trained models (ResNet, MobileNet)

Tools

PyTorch or TensorFlow/Keras

Practice

Load a pre-trained ResNet18

Run inference on sample images

Mini Project
Image Classifier – Use MobileNet to classify 10 custom test images.

Day 5: Object Detection
Topics

Sliding windows vs region proposals

YOLO, SSD, Faster R-CNN basics

Using a pre-trained YOLO (v8 or v5)

Practice

Detect objects in a video frame

Draw bounding boxes & labels

Mini Project
Real-Time Person Counter – Detect people in a webcam feed and count them.

Day 6: Image Segmentation & Face Recognition
Topics

Semantic vs instance segmentation (UNet, Mask R-CNN)

Face detection (Haar cascades, MTCNN)

Face recognition (face_recognition library, DeepFace)

Practice

Segment foreground/background with a pre-trained model

Detect and blur faces in an image

Mini Project
Face Swapper – Swap faces between two images using landmarks + alignment.

Day 7: End-to-End Project + Deployment
Topics

Model optimization (quantization, pruning)

Deploying with Flask / FastAPI / Streamlit

Edge deployment (brief intro to TensorFlow Lite, ONNX)

Final Project (choose one)

Option A: Defect Detection for manufacturing (custom dataset + YOLO)

Option B: Hand Gesture Controller (track hand landmarks → control volume or mouse)

Option C: Traffic Sign Recognizer (train small CNN on GTSRB dataset)

Deliverable

Working demo (local app or web interface)

GitHub repo with README, code, and sample outputs

Resources per Day
Day	Key Libraries	Datasets (for practice)
1-3	OpenCV, NumPy, Matplotlib	Own images, COCO, VOC
4	PyTorch/TF, torchvision	ImageNet sample, CIFAR-10
5	Ultralytics YOLO	COCO subset, Open Images
6	dlib, face_recognition	LFW, CelebA, FDDB
7	Streamlit, ONNX	Custom collected data or GTSRB
Daily Schedule (≈3–4 hours)
Theory (45 min) – Video or article from the topic list.

Coding practice (1.5 hours) – Implement core functions.

Mini project (1 hour) – Apply to a real image/video.

Log your results (15 min) – Save outputs, note challenges.

After Week 7 – Next Steps
Try Kaggle competitions (e.g., Humpback Whale Identification)

Learn Vision Transformers (ViT)

Build a multi-model pipeline (detection → tracking → action recognition)
