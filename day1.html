🎯 Mini Project: Image Color Extractor

🎨 Extract dominant color channel & display histograms

Build a tool that loads any image, extracts individual Red, Green, Blue channels, and visualizes their intensity histograms. Identify which color channel dominates the image's look.

✨ Expected features:
  • Read image (support .jpg, .png)
  • Compute histogram for each channel (256 bins)
  • Plot original image + 3 histograms (R/G/B) on a single figure
  • Optional: print which channel has highest average pixel intensity
# Pseudocode / core logic
import cv2, matplotlib.pyplot as plt
img = cv2.imread('sample.jpg')
img_rgb = cv2.cvtColor(img, cv2.COLOR_BGR2RGB)
colors = ('red','green','blue')
for i, col in enumerate(colors):
   hist = cv2.calcHist([img_rgb], [i], None, [256], [0,256])
   plt.plot(hist, color=col)
plt.title('RGB Channel Histograms')
plt.show()
