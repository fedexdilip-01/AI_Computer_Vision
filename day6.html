📖 Image Segmentation: What, Why, How, Where

❓ WHAT is Segmentation?

Semantic: Classify each pixel (e.g., road, sky, person). Instance: Separate individual objects (person1, person2, car1). Output = mask overlay.

🖼️ Original → 🎨 Mask (pink=person, blue=sky)
🎯 WHY Segmentation?

Autonomous driving (road/lane detection), medical imaging (tumor segmentation), background removal, photo editing, robotics scene understanding.

⚙️ HOW: UNet & Mask R-CNN

UNet: Encoder-decoder with skip connections — great for biomedical segmentation. Mask R-CNN: Extends Faster R-CNN with pixel-level masks.

🌍 WHERE applied?

Satellite imagery, self-driving cars, virtual backgrounds (Zoom/Teams), AI photo editing, crop disease detection.

🧪 Key difference: Detection = bounding boxes. Segmentation = pixel‑perfect boundaries. Mask R‑CNN does both!

🖍️ Semantic vs Instance Segmentation

🎨 Semantic Segmentation
All "cars" → same color mask
[🟦 sky] [🟩 grass] [🟥 car]
👥 Instance Segmentation
Car A → mask 1, Car B → mask 2
Differentiates individual objects

👤 Face Detection: Haar Cascades & MTCNN

📐 Haar Cascades (OpenCV)

Classic, fast, CPU-friendly. Uses cascaded features. Works well for frontal faces. Limited with angles/occlusions.

face_cascade = cv2.CascadeClassifier(cv2.data.haarcascades + 'haarcascade_frontalface_default.xml')
faces = face_cascade.detectMultiScale(gray, scaleFactor=1.1, minNeighbors=5)
🧠 MTCNN (Multi-task CNN)

Deep learning based. More accurate, handles profile faces, lighting variations. Returns bounding boxes + facial landmarks (eyes, nose, mouth).

from mtcnn import MTCNN
detector = MTCNN()
faces = detector.detect_faces(img)
# each face: box, keypoints (left_eye, right_eye, nose, mouth_left, mouth_right)
💡 Practice: Detect faces in a group photo with both methods. MTCNN handles side profiles better, Haar is faster for real-time on CPU.

🔍 Face Recognition: Who is this person?

📦 face_recognition library

Simple API, built on dlib. Face encoding (128-dim vector) → compare encodings to identify known faces.

import face_recognition
image = face_recognition.load_image_file("person.jpg")
encoding = face_recognition.face_encodings(image)[0]
matches = face_recognition.compare_faces(known_encodings, encoding)
🧠 DeepFace

Wrapper for state-of-the-art models (VGGFace, FaceNet, ArcFace). Handles verification, recognition, and attribute analysis (age, gender, emotion).

from deepface import DeepFace
result = DeepFace.verify(img1_path, img2_path) # True/False
embedding = DeepFace.represent(img_path, model_name='Facenet')[0]["embedding"]

🔄 Mini Project: Face Swapper

🎭 Swap faces between two images using landmarks + alignment

Goal: Take two faces (source & target) → detect facial landmarks → align source face to target orientation → seamless blend.

1
Detect faces & landmarks – Use dlib or MTCNN to get 68 facial landmarks (eyes, nose, jawline).
2
Compute convex hull & Delaunay triangulation – Create triangle mesh covering the face region.
3
Warp triangles (affine transform) – Transform each triangle from source to target face geometry.
4
Seamless cloning (color correction & blending) – Use OpenCV's seamlessClone to blend edges, match skin tone.
5
Output swapped image – Source face now on target's head!
# Face Swapper using dlib + OpenCV (simplified pipeline)
import cv2, dlib, numpy as np

detector = dlib.get_frontal_face_detector()
predictor = dlib.shape_predictor("shape_predictor_68_face_landmarks.dat")

def get_landmarks(img):
faces = detector(img, 1)
if len(faces) == 0: return None
return np.array([[p.x, p.y] for p in predictor(img, faces[0]).parts()])

def face_swap(source_img, target_img):
src_pts = get_landmarks(source_img)
tgt_pts = get_landmarks(target_img)
if src_pts is None or tgt_pts is None: return target_img

# Compute convex hull and triangulation
hull_src = cv2.convexHull(src_pts)
hull_tgt = cv2.convexHull(tgt_pts)

# Warp source face to target geometry
warped_face = warp_triangles(source_img, src_pts, tgt_pts, hull_src, hull_tgt)

# Create mask and seamless clone
mask = np.zeros(target_img.shape[:2], dtype=np.uint8)
cv2.fillConvexPoly(mask, hull_tgt, 255)
output = cv2.seamlessClone(warped_face, target_img, mask, (tgt_pts[30][0], tgt_pts[30][1]), cv2.NORMAL_CLONE)
return output

# Usage:
source = cv2.imread("face_a.jpg")
target = cv2.imread("face_b.jpg")
swapped = face_swap(source, target)
cv2.imwrite("face_swapped.jpg", swapped)
🎭 [SOURCE FACE] + [TARGET HEAD] → 🔄 Landmark alignment → ✨ Swapped face result
🧪 Try it yourself: Use two portrait photos (front-facing works best). The landmark-based warping preserves expression and orientation.
⚡ Pro tip: For better skin blending, apply color correction (histogram matching) before seamlessClone.

📌 Best Practices for Segmentation & Face Recognition

  • 🔹 For face detection: MTCNN is more robust (angles/lighting), Haar is faster for real-time CPU apps.
  • 🔹 Face recognition accuracy: DeepFace (Facenet/ArcFace) > face_recognition. Choose based on required precision.
  • 🔹 Segmentation models: UNet for medical/binary masks, Mask R-CNN for instance segmentation (80 COCO classes).
  • 🔹 Face swapping: Always align landmarks and use seamless cloning for realistic skin blending.
  • 🔹 Privacy: Face blurring is essential for public CCTV/datasets compliance (GDPR).
